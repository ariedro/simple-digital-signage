# Simple Digital Signage

This is an example of a Digital Signage software done in NodeJS

## Install


1. Install [NodeJS v12](https://nodejs.org/en/download/) with `npm`
2. Install the dependencies

```
npm install
```

## Run in development mode

```
npm run dev
```

## Run tests

```
npm run test
```
